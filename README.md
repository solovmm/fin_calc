# Fin_calc (MVP)

## Что внутри
- index.html — главная страница выбора калькуляторов
- inflation.html — инфляционный калькулятор (данные из inflation_ru_full_1991_2024.json)
- ndfl.html — калькулятор НДФЛ (правила в ndfl_rules.json)
- assets/ — общий стиль и скрипты

## Как открыть локально
Самый простой способ:
- Откройте папку в VS Code
- Поставьте расширение Live Server
- Запустите `index.html`

## Как залить на GitHub Pages
Положите файлы в корень репозитория и включите Pages:
Settings → Pages → Deploy from branch → main / root
