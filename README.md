# Inflation calculator (static)
Файлы:

Важно: расчет выполняется только после нажатия кнопки «Рассчитать». После любых изменений появится статус «Изменено. Нажмите „Рассчитать“».
- `index.html` — интерфейс
- `app.js` — логика
- `inflation_ru_full_1991_2024.json` — данные

## Как открыть локально
```bash
python3 -m http.server 8000
```
Открыть: http://localhost:8000/index.html
